# vuex-demo

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Customize extensition
1. ESLint
2. Vetur
3. Vue VSCode Snippets
4. Vuter

### Package required
```
npm install vuex
npm install axios
npm install vue-uuid
```
